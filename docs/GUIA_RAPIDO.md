## 🔐 Como Obter Acesso

### Opção 1: Conta de Teste (Para Experimentação)

**Para quem:** Pessoas que querem conhecer o sistema antes de usar oficialmente

**Como obter:** 
- Fale com o **Desenvolvedor** para receber credenciais temporárias
- Você poderá explorar todas as funcionalidades sem compromisso

**Duração:** Temporária (até criar conta oficial)

---

### Opção 2: Conta Oficial (Para Uso Diário)

**Para quem:** Pessoas que vão usar o sistema regularmente

**Como obter:**
- Envie um **e-mail válido** para o desenvolvedor, OU
- Procure a **secretaria da escola** para criar seu perfil oficial

**Importante:** 
- Use um e-mail que você acessa regularmente
- Este e-mail será sua identificação no sistema
- Você receberá uma senha de acesso

---

## 👥 Perfis de Acesso

Cada pessoa terá um perfil conforme seu papel na escola:

| Perfil | Quem Usa | O que Pode Fazer |
|--------|----------|------------------|
| 👨‍🎓 **Aluno** | Estudantes matriculados | Ver notas, solicitar documentos, ver matriz curricular |
| 👨‍🏫 **Professor** | Docentes | Lançar notas, registrar aulas, ver frequência dos alunos |
| 🏫 **Secretaria/Admin** | Equipe gestora | Acesso total: turmas, matrículas, cadastros, documentos |

## 📚 Treinamento

### Onde Acessar

O treinamento está disponível em **4 locais**:

1. **Tela de Login** → Botão amarelo "📚 Material de Treinamento"
2. **Página Inicial** → Opção "📚 Material de Treinamento"
3. **Dentro do Sistema** (após login) → Link "📚 Treinamento" na barra lateral
4. **Diretamente** → Abra o arquivo `apresentacao_treinamento.html` no navegador

### O que o Treinamento Inclui

- ✅ 15 slides interativos com navegação fácil
- ✅ Explicações didáticas para adultos
- ✅ Esquema de cores do Colégio Santa Mônica (Vermelho, Amarelo, Cinza, Branco)
- ✅ Funcionalidades de cada perfil
- ✅ Permissões e restrições
- ✅ Passo a passo de cada operação
- ✅ Dicas importantes para cada perfil
- ✅ Sistema de cálculo de notas
- ✅ Fluxo de solicitação de documentos
- ✅ Segurança e proteção de dados

**Tempo estimado para revisar:** 15-20 minutos
