📚 O que é o Sistema de Gestão Escolar?

O SGE é uma plataforma digital desenvolvida especialmente para o Colégio Santa Mônica que permite gerenciar todas as atividades acadêmicas de forma organizada, segura e eficiente.

🎯 Objetivos do Sistema

  • Centralizar informações acadêmicas em um único lugar
  • Facilitar a comunicação entre alunos, professores e secretaria
  • Automatizar processos como lançamento de notas e emissão de documentos
  • Garantir segurança e privacidade dos dados
  • Permitir acesso de qualquer lugar com internet

✨ Principais Vantagens

🌐 Acesso Online

Acesse de qualquer computador ou celular com internet

🔒 Segurança

Dados protegidos com criptografia e controle de acesso por perfil

⚡ Automatização

Cálculo automático de médias e geração de documentos PDF

📊 Organização

Todas as informações organizadas e fáceis de encontrar

📊 Sistema de Cálculo Automático de Notas

O sistema calcula automaticamente as médias dos alunos:

  • Média Teórica: (N1 + N2 + N3) ÷ 3
  • Média Final: (Média + Recuperação) ÷ 2 (se houver recuperação)

Status: ✅ Aprovado (≥ 7.0) | 🟡 Recuperação (5.0-6.9) | 🔴 Reprovado (< 5.0)

🎓 Gestão de Turmas e Matrículas

A gestão de turmas é uma funcionalidade essencial para organização da escola, disponível apenas para Secretaria e Admin.

📝 Criando uma Nova Turma

  1. No menu lateral, clique em "Gestão de Turmas"
  2. Preencha o campo "Nome da Turma" (ex: Técnico Enfermagem Manhã - 10A)
  3. Preencha o campo "Período Letivo" (ex: 2026.1)
  4. Clique em "Registrar Turma"

👨‍🎓 Matriculando Alunos em uma Turma

  1. Clique na turma desejada na lista à esquerda
  2. No campo "Matricular Aluno Existente", selecione o aluno
  3. Clique em "Adicionar à Turma"

📊 Status Acadêmico dos Alunos

Status Descrição Quando Usar
Ativo Aluno regular Aluno frequentando aulas normalmente
Trancado Matrícula trancada Aluno trancou temporariamente
Evadido Abandonou o curso Aluno não retorna mais às aulas
Concluído Formado com sucesso Aluno completou todas as disciplinas
⚠️ Atenção: NUNCA exclua um aluno da turma! Sempre altere o status para "Trancado" ou "Evadido" para preservar o histórico acadêmico.

📊 Lançamento de Notas

O lançamento de notas é uma das funções mais importantes para os professores. O sistema facilita o processo com cálculo automático.

📝 Como Lançar Notas

  1. No menu lateral, clique em "Minhas Turmas"
  2. Selecione a disciplina desejada
  3. O sistema mostrará todos os alunos da turma automaticamente
  4. Para cada aluno, preencha:
    • Faltas: Quantidade de ausências
    • N1: Primeira nota (0 a 10)
    • N2: Segunda nota (0 a 10)
    • N3: Terceira nota (0 a 10)
    • Rec.: Nota de recuperação (se necessário)
  5. Clique em "Salvar Todas as Notas" para salvar de uma vez

💡 Dicas Importantes para Professores

  • Registre as notas o mais rápido possível após as provas
  • Verifique as faltas antes de salvar, pois afetam a aprovação
  • Use a recuperação apenas para alunos com média abaixo de 7.0
  • Você pode exportar notas em PDF para seu controle

🎨 Interpretação das Cores

🟢 Verde

Aprovado: Média Final ≥ 7.0

🟡 Amarelo

Recuperação: Média Final entre 5.0 e 6.9

🔴 Vermelho

Reprovado: Média Final < 5.0

📅 Registro de Aulas

O registro de aulas permite documentar o conteúdo ministrado em cada aula, criando um histórico organizado para o professor e para a escola.

📝 Como Registrar uma Aula

  1. No menu lateral, clique em "Registrar Aula"
  2. No campo "Selecione a Disciplina", escolha a matéria
  3. Preencha o formulário:
    • Data: Data da aula (já vem preenchida com a data de hoje)
    • Conteúdo Ministrado: Descreva o que foi ensinado
  4. Clique em "Registrar"

📋 Visualizando o Histórico

Abaixo do formulário, você verá todas as aulas já registradas para essa disciplina:

  • Data da aula
  • Conteúdo ministrado
  • Botão "Remover" (para excluir registros incorretos)

💡 Exemplo de Registro

Data: 01/04/2026

Conteúdo: Introdução à Anatomia Humana - Sistema Esquelético

🎯 Por que registrar aulas?

  • Mantém um histórico organizado do conteúdo ministrado
  • Facilita o acompanhamento do progresso da turma
  • Ajuda em caso de substituição de professor
  • Documenta o cumprimento do plano de ensino

📄 Solicitação de Documentos

O sistema permite solicitar e gerar documentos acadêmicos de forma automatizada com o cabeçalho oficial do Colégio Santa Mônica.

📋 Tipos de Documentos Disponíveis

Tipo de Documento Quem Pode Solicitar O que é Gerado
Declaração de Matrícula Aluno PDF com dados do curso e turma
Histórico Acadêmico Aluno PDF com todas as notas e situação
Atestado de Frequência Aluno PDF com frequência do aluno
Declaração de Vínculo Professor / Admin / Secretaria PDF com função exercida na escola

📝 Fluxo de Solicitação

  1. Aluno solicita o documento pelo menu "Documentos"
  2. Secretaria vê o pedido no Painel da Secretaria → Solicitações
  3. Secretaria clica em "Gerar PDF" para criar o documento
  4. O PDF é baixado automaticamente para o computador
  5. Secretaria imprime e entrega ao solicitante
  6. Status muda para "Concluído"
💡 Dica: O sistema gera automaticamente o PDF correto baseado no perfil do solicitante. Alunos recebem documentos acadêmicos, enquanto funcionários recebem declarações de vínculo.

🔒 Segurança e Proteção de Dados

O sistema implementa diversas camadas de segurança para proteger os dados dos usuários:

🛡️ Proteções Implementadas

🔐 Autenticação Segura

Login com e-mail e senha criptografados. Session timeout automático após 30 minutos de inatividade

🚫 Proteção contra Força Bruta

Após 5 tentativas de login erradas, a conta é bloqueada por 15 minutos

👤 Controle de Acesso por Perfil

Cada usuário vê apenas o que tem permissão. Alunos não acessam dados de outros alunos

🔒 Criptografia de Dados

Todos os dados são transmitidos com criptografia HTTPS

📊 Row Level Security (RLS)

O sistema usa políticas de segurança em nível de banco de dados:

Tabela Quem pode ler Quem pode escrever
Perfis Próprio usuário, Admin Próprio usuário, Admin
Notas (Boletim) Aluno (só suas), Professor, Admin Professor, Admin
Aulas Professor (só suas), Admin Professor
Matrículas Próprio aluno, Admin Admin

💡 LGPD - Lei Geral de Proteção de Dados

O sistema está em conformidade com a LGPD. Os dados dos alunos e professores são tratados com sigilo e utilizados apenas para fins acadêmicos.
